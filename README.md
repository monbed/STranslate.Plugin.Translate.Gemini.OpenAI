# STranslate.Plugin.Translate.Gemini.OpenAI

一个为 [STranslate](https://github.com/ZGGSONG/STranslate) 应用提供的 Gemini(OpenAI兼容) 翻译插件。

## 快速开始

### 安装

1. 下载最新版本的.spkg 插件包
2. 打开 STranslate 软件，进入 设置 > 插件。将下载的文件拖入窗口即可安装。

### 配置

首次使用前需要进行如下配置：

1. 打开 STranslate 应用
2. 添加 Gemini(OpenAI兼容) 作为翻译引擎
3. 配置必要参数：
   - **接口**：使用官方不需要修改，如需使用第三方平台Path为 /v1/chat/completions
   - **API Key**：Gemini API 密钥
   - **模型选择**：选择要使用的 Gemini 模型
   - **其他参数**：根据需求调整温度等参数
